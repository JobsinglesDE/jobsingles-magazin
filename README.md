# GastroSingles Magazin

Redaktionelles Online-Magazin von **GastroSingles** – der Dating-Plattform für Menschen aus Gastronomie, Küche und Hotellerie. Ratgeber, Promiköche-Porträts und Partnersuche-Tipps für die Branche.

Gebaut mit [Next.js](https://nextjs.org/) und [Keystatic](https://keystatic.com/), deployed auf [Vercel](https://vercel.com/).
